Creating an Angular 2 component library - Workshop

by Raul Jimenez @elecash

Angular 2 is the next evolution to the AngularJS framework and now the Beta release is out!

With the new year it's a perfect time to start a side project and learn new things. Angular 2 comes with its own challenges like TypeScript compilation and functional programming.

Come to learn how to deal with these new framework. We will focus in the creation of an Angular 2 reusable component, scoped styles and templates, publication and distribution to NPM.



About Raul Jimenez

Raul is a software engineer passionate about JS frameworks, web technologies and cross-platform development. Since 2012 he's focused in Angular, HTML5 video and his open source project Videogular.
Raul works as a freelance front-end architect for companies around the world helping them to build high-performance web apps. In his spare time he's usually involved in local meetups, writing blog posts, speaking at conferences and contributing to open source projects.
He lives in Barcelona and is happy by default

About
